Customer Shopping Behaviour Analysis

📌 Project Overview

This project analyzes customer shopping behavior using Python.
The goal is to understand purchasing patterns, customer preferences, and trends.

📂 Dataset

The dataset is in CSV format and contains:

- Customer Age
- Gender
- Product Category
- Purchase Amount
- Payment Method

🛠 Tools & Technologies

- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook / VS Code

📊 Analysis Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization of customer trends
- Category-wise purchase analysis

📈 Key Insights

- Identified most popular product categories
- Analyzed spending patterns by gender
- Found trends in customer age groups

▶️ How to Run

1. Install required libraries:
   pip install pandas matplotlib seaborn

2. Run the Python file:
   python analysis_1.py
